## Generation Tips

- Generate all 5 in one batch (Midjourney or Canva AI) so light and base tone stay consistent.
- In Canva: build Board 1 first, lock the color scheme, then duplicate the template for boards 2-5.
- If using Midjourney: add `--ar 1:1 --style raw` and reference the first output as `--cref` for character/style consistency across the set.
- No bottom bar, no URL on board covers — keep them clean. The branding comes from the palette and typography alone.
