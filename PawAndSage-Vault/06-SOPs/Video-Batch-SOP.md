# SOP: Saturday Video Batch Production

## Prep (10 min)
1. Pick 5 articles from Content Plan
2. For each: write the hook (first 3 seconds) and key points
3. Note which video type: Explainer, Listicle, Product, Quick Tip, Myth Bust

## Production (60-90 min)
1. Open Pexels → search stock cat videos → download 1-2 clips per video
2. Open CapCut → duplicate your template
3. For each video:
   - Drop in stock clip
   - Add text overlays (hook, key points, CTA)
   - Add background music (CapCut library, lo-fi or chill)
   - Export as 9:16, 1080p
4. Save all 5 videos to phone

## Scheduling (15 min)
1. Open Metricool
2. Upload all 5 videos
3. For each video:
   - Write caption (2-3 sentences + 3 hashtags)
   - Select platforms: TikTok + Instagram Reels + YouTube Shorts
   - Schedule: Mon through Fri, one per day
4. Done — they post automatically all week

## Post-Batch
- Update Video Plan in vault
- Note which articles have been turned into videos
